# 飞书排版规范（三任务共用）

飞书自定义机器人发的是 **纯文本**，不要用 Markdown 表格。  
三段任务统一用下面结构，只改标题与时段字段。

## 通用规则

1. 全角分隔线：`────────`
2. 区块标题用：`【大盘】` `【热点】` `【标的】` `【策略】` `【节奏】` `【仓位】` `【置顶】`
3. 每行一个信息点；热点/标的各占一行，勿挤成一团
4. 数字尽量短：`沪+1.2%`、`额2.7万亿`、`排#12`
5. 精简版总行数：早盘/午盘 ≤22 行；盘后 ≤26 行
6. 只发 **一条** 消息（整段一个字符串）
